Quadratic Programming Portfolio Optimisation

This project implements portfolio optimisation using a Quadratic Programming (QP) approach based on the classical Markowitz mean–variance framework from Operations Research. The objective is to determine the optimal allocation of investment among a set of NSE stocks such that the overall portfolio risk is minimised for a given level of expected return.

Problem Description

Let x denote the vector of portfolio weights, μ denote the vector of expected returns and Σ denote the covariance matrix of asset returns. The optimisation problem considered in this project is:

Minimise
xᵀ Σ x

Subject to
μᵀ x = T
∑ xᵢ = 1
xᵢ ≥ 0 for all assets

Here, T represents the target expected return chosen by the investor. Since the covariance matrix is positive semi-definite, the objective function is convex and hence the obtained solution is globally optimal. The constrained quadratic programming problem is solved using the Karush–Kuhn–Tucker (KKT) optimality conditions which reduce the problem to solving a system of linear equations.

Data

Daily closing price data of selected NSE stocks is downloaded using the yfinance Python library. From the price data, daily returns are computed and used to estimate the expected return vector and covariance matrix required for the optimisation model.

Project Workflow

Stock price data is downloaded and stored as CSV files.
Daily returns, expected returns and covariance matrix are computed.
The quadratic programming model is formulated and solved using KKT conditions.
The optimal portfolio weights are obtained for a chosen target return.
An efficient frontier is generated by solving the model for multiple target returns.
Results including optimal weights, summary statistics and plots are saved in the results folder.

How to Run

Step 1 – Download stock data
python src/download_data.py

Step 2 – Compute statistics
python src/compute_stats.py

Step 3 – Solve optimisation model
cd src
python qp_solver.py

Step 4 – Generate efficient frontier
python frontier.py
python plot_frontier.py

Output

The program produces the optimal portfolio allocation along with expected return, variance and volatility of the portfolio. It also generates the efficient frontier plot which illustrates the trade-off between risk and return. All outputs are saved inside the results directory.

Assumptions

The model assumes long-only investment positions and ignores transaction costs, taxes and liquidity constraints. Historical returns are used as an estimate of future expected returns.

This project demonstrates the application of quadratic programming techniques from Operations Research to practical financial portfolio optimisation problems.
